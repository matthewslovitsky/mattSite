COLLEGE EXPERIENCE

While studying Computer Science at Siena College in Loudonville, New York, I developed a strong passion for software development. I dedicated my time to learning new programming languages and building projects that deepened my understanding and skills in the field.

CODE KNOWLEDGE

At Siena, our coding curriculum began with Java, providing a strong foundation in core programming concepts such as inheritance, abstraction, and interfaces.
Python quickly became one of my favorite languages due to its readability and versatility, which I used for data analysis, algorithm design, and automation projects.
HTML served as the foundation of my web development journey, helping me create accessible and semantic websites.
JavaScript added dynamic interactivity to my websites, and I used it extensively in front-end and full-stack projects.
With CSS, I styled layouts using modern systems like Flexbox and Grid to ensure responsive design.
I used PHP for server-side scripting, database interactions, and rendering dynamic web content.

FRAMEWORKS & TOOLS

GitHub allowed me to manage version control, conduct code reviews, and collaborate on team-based software projects.
With Bootstrap, I built responsive interfaces using its grid system and prebuilt components to streamline development.
npm helped me manage JavaScript dependencies and automate development workflows in Node.js applications.
Composer was my go-to dependency manager for PHP, enabling modular backend development with Laravel.
VSCode became my primary code editor, offering extensions, Git tools, and live debugging to streamline development.
Figma supported my UI/UX design work with its collaborative tools, wireframing, and interface prototyping capabilities.
Photoshop helped me edit and create visual assets, ensuring polished, professional-looking design elements.
Excel was essential for data analysis, planning, and tracking metrics through formulas and visualizations.
I worked extensively with SQLite to manage lightweight, file-based relational databases. Its simplicity made it ideal for prototyping and embedded systems.
Laravel helped me structure web apps with MVC, manage routing, and interact with databases using Eloquent ORM.
With Express.js, I built lightweight RESTful APIs and backend services for full-stack JavaScript applications.
